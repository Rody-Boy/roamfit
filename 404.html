RoamFit is a credit-based fitness marketplace connecting members with independent gyms and fitness facilities. This GitHub Pages site is the public landing page while the Laravel MVP application is prepared for PHP hosting.

GitHub Pages can only serve static files. The Laravel app in this repository requires a PHP host such as Laravel Forge, Ploi, Render, Railway, DigitalOcean, or another server that can run PHP and a database.
